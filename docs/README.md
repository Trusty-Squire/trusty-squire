# Trusty Squire Docs

This directory contains the public documentation that should stay current for a
launch reader.

- [ARCHITECTURE.md](ARCHITECTURE.md) is the canonical project document. Start
  there for the product model, security boundaries, app layout, provisioning
  flow, and skill lifecycle.
- [VAULT-OPERATIONS.md](VAULT-OPERATIONS.md) is the operator runbook for the
  credential vault.
- [DEPLOY-registry.md](DEPLOY-registry.md) covers deploying the skill registry.
- [BUSINESS-MODEL.md](BUSINESS-MODEL.md) records the current pricing and
  positioning model.

Historical design notes and obsolete launch plans are intentionally not kept in
the public docs tree. They remain available through git history.
